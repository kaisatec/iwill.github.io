使用 Hexo 搭建 GitHub Pages

2009 年 Zigzag 就教了我怎么用 GitHub Pages 写博客，可是 …. 他去天堂写代码已经快满 4 年了，Zig 我对不起你！为了避免被我对不起的 Zig 带走，周末开始搭建自己的博客了。

看到我订阅的博客多半用的是 Octopress，所以认定这一定是个高大上。但是被总是安装不成功，貌似是升级 XCode 5.1.1 导致一些东西编译不过，非 Ruby Fan 无法坚持了。最终我选了 Hexo [hekso]，A fast, simple & powerful blog framework, powered by Node.js，与前者类似 —— 将 Markdown 转成用于发布的 HTML。
